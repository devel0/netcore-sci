Detailed Description

comparer to sort angles so that they follow runs from Start and End given. for example giving start=5/4PI, end=PI/2 then the set PI/4 and 3/4PI will sorted as 3/4PI and PI/4. Precondition: at constructor and as input elements angles must already normalized [0,2PI) and satisfy they in normalized range between given start, end angles
