Class MUDomain

list of physical quantities

Measures here contains information about implicit measure unit and value of the tolerance.

Note that all measure must be dimensionally equivalent. For example: [length] = m [length2] = [length] * [length] = m2 [time] = s [time2] = [time] * [time] = s2 [speed] = [length] / [time] = m/s [acceleration] = [length] / [time2] = m/s2 [mass] = kg [force] = [mass] * [acceleration] = kg * m/s2 = N [pressure] = [force] / [length2] = N / m2 = Pa

This will ensure measure comparision without further conversion, for example m1 = 1 [kg] a1 = 2 [m/s2] f1 = 4 [N]

test = m1 * a1 > f1
